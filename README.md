# 🏥 Health App – Patient Management System

A modern, full-stack patient management system built with **Next.js (App Router)**, **MongoDB**, and **Tailwind CSS**. This application allows hospitals or clinics to manage patient records, track treatment stages, update medical information, and handle emergency contacts through a clean, responsive dashboard.

---

##  Features

-  Secure login & logout system
-  Add, view, update, and manage patient records
-  Track treatment stages (Diagnosis → Treatment → Recovery → Discharge)
-  Manage medications and allergies
-  Emergency contact management
-  Admission & discharge planning
-  Dark mode UI with glassmorphism design
-  Fully responsive dashboard

---

##  Tech Stack

- **Frontend & Backend:** Next.js (App Router), Three.js , framer motion
- **Styling:** Tailwind CSS
- **Database:** MongoDB (via Mongoose)
- **Icons:** Lucide React
- **State Management:** React Hooks
